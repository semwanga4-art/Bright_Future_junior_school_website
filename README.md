# README - Bright Future Junior School Website

## Project Overview
This is a comprehensive website for **Bright Future Junior School**, an educational institution dedicated to providing quality education for children aged 3-14 years. The website features a modern, responsive design with multiple pages showcasing the school's programs, values, and community.

## Project Structure
```
project-folder/
│
├── index.html          # Homepage
├── about.html          # About Us page
├── services.html       # Educational Programs page
├── contact.html        # Contact Us page
├── blog.html           # School News & Announcements
├── style.css           # Main stylesheet
└── Asserts/            # Assets directory
    ├── log.jpg         # School logo
    ├── building.jpeg   # School building image
    ├── building1.jpg   # Additional building image
    ├── winners.jpg     # Science fair winners image
    └── [other images]  # Additional blog images
```

## Features

### Design & User Experience
- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean, professional design with consistent color scheme
- **Smooth Animations**: CSS transitions and hover effects
- **Mobile-Friendly Navigation**: Hamburger menu for mobile devices

### Pages & Content
1. **Homepage** (`index.html`)
   - Hero section with school mission statement
   - About preview section
   - Educational programs overview
   - Call-to-action buttons

2. **About Us** (`about.html`)
   - School background and history
   - Mission, vision, and values
   - School building image

3. **Services/Programs** (`services.html`)
   - Detailed educational programs:
     - Early Years (3-5 years)
     - Lower Primary (6-8 years)
     - Upper Primary (9-12 years)
     - Extracurricular activities
     - After school program
     - Special needs support

4. **Contact Us** (`contact.html`)
   - Contact information (address, phone, email)
   - Contact form with validation
   - Office hours

5. **Blog/News** (`blog.html`)
   - School announcements and events
   - Latest news articles
   - Image gallery of school activities

### Technical Features
- **CSS Variables**: Consistent color scheme and styling
- **Grid & Flexbox Layouts**: Modern CSS layout techniques
- **Form Handling**: JavaScript form submission
- **Mobile Menu**: JavaScript-powered hamburger menu
- **Cross-browser Compatibility**: Works on all modern browsers

## Color Scheme
The website uses a consistent color palette defined in CSS variables:
- Primary Color: `#3498db` (Blue)
- Secondary Color: `#2c3e50` (Dark Blue)
- Accent Color: `#e74c3c` (Red)
- Light Color: `#ecf0f1` (Light Gray)
- Text Colors: Various shades for hierarchy

## Typography
- **Primary Font**: 'Poppins' - Used for body text
- **Heading Font**: 'Roboto Slab' - Used for titles and headings

## Browser Compatibility
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Setup Instructions

1. **Download Files**: Ensure all HTML, CSS, and image files are in the same directory
2. **File Structure**: Maintain the folder structure as shown above
3. **Web Server**: For best performance, serve through a local web server
4. **Testing**: Open `index.html` in a web browser to view the website

## Customization

### Updating School Information
- Edit contact details in `contact.html`
- Modify school information in `about.html`
- Update programs in `services.html`

### Styling Changes
- Modify CSS variables in `:root` section of `style.css`
- Adjust colors, fonts, and spacing as needed

### Adding New Content
- Follow the existing HTML structure for consistency
- Maintain the CSS class naming convention
- Ensure responsive design is preserved
